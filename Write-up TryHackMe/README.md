# TryHackMe Write-ups

## Overview
This repository contains a curated collection of write-ups based on hands-on labs, scenarios, and challenges completed on the TryHackMe platform.
It is part of my journey toward becoming a SOC Analyst, with a focus on developing practical skills in threat detection, incident analysis, and security operations.

## Objectives
- Demonstrate practical cybersecurity skills through real-world scenarios
- Build a structured and documented learning path
- Strengthen analytical and problem-solving abilities
- Showcase familiarity with common SOC tools and methodologies

## Repository Structure
Each room or challenge is organized into its own dedicated folder, containing a detailed write-up.

Each write-up typically includes:

- Scenario overview
- Methodology and approach
- Tools and commands used
- Analysis and findings
- Key takeaways

## Skills Developed
Through these exercises, I am actively developing:

- Log analysis and event investigation
- Network traffic analysis
- Threat detection and response
- Basic digital forensics
- Use of security tools (e.g., SIEM, Wireshark, Linux utilities)

## Disclaimer
These write-ups are intended for educational purposes only and may contain spoilers for TryHackMe content.

## Continuous Improvement
This repository is continuously updated as I progress through new challenges and refine my skills.
