# Wireshark Labs

This section contains hands-on network traffic analysis projects performed using Wireshark. The goal of these labs is to develop practical skills in packet analysis, protocol inspection, and basic threat detection.


## Labs Included
- HTTP Traffic Analysis
- DNS Analysis
- HTTPS / TLS Analysis
- Suspicious Traffic Investigation

Each lab includes:

- A clear objective
- Methodology
- Detailed analysis
- Key findings
- Supporting screenshots


## Skills Demonstrated
- Network protocol analysis (HTTP, DNS, TCP, TLS)
- Packet inspection and filtering
- Identifying unencrypted traffic
- Basic threat detection techniques


## Tools Used
- Wireshark
