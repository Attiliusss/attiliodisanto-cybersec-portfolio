# Malicious Traffic Analysis

This directory is part of my cybersec-portfolio and showcases my hands-on practice in analyzing suspicious and malicious network traffic.


## Purpose
As an aspiring SOC Analyst (Level 1), I created this section to document my learning journey and demonstrate my practical skills in:
- Network Traffic Analysis
- Threat Detection & Investigation
- PCAP analysis
- Identifying malicious patterns and behaviors


## Lab Content
Each Lab includes:
- Scenario description
- Analysis objectives
- Tools used (e.g., Wireshark, Zeek, tcpdump)
- Step-by-step investigation process
- Key findings and evidence
- Final assessment and conclusions


## Screenshots
Every Lab contains a dedicated screenshots/ folder, where I collect visual evidence of my analysis.
This helps provide clear documentation of my workflow and supports my findings.
