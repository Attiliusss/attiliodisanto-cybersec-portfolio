# LetsDefend Write-ups

## Overview
This repository contains a curated collection of write-ups based on investigations, alerts, and challenges completed on the LetsDefend platform.
It is part of my journey toward becoming a SOC Analyst, with a focus on developing practical experience in security monitoring, incident investigation, threat detection, and blue team operations.


## Objectives
- Demonstrate practical SOC and Blue Team skills through realistic security scenarios
- Build a structured and documented learning path
- Improve analytical and investigative techniques
- Showcase familiarity with security operations workflows, tools, and methodologies


## Repository Structure
The repository is organized into two main sections:

### Alert Investigations
This section contains write-ups of alert investigations from the **Practice** area of LetsDefend.

Each investigation typically includes:
- Alert overview
- Initial triage
- Log analysis
- Indicators of Compromise (IOCs)
- Investigation process
- Root cause analysis
- Final verdict and response
- Key takeaways

### Challenges
This section contains solutions and write-ups for LetsDefend challenges.

Each challenge typically includes:
- Challenge overview
- Methodology and approach
- Tools and techniques used
- Analysis and findings
- Lessons learned


## Skills Developed

- Security event monitoring

- Log analysis and correlation
- Threat detection and incident response
- IOC identification and analysis
- Network and endpoint investigation
