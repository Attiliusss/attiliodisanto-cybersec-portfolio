# What I've learned

In my opinion, the two most challenging levels in Bandit were:

- Level25: Initially, I wasn’t familiar with pagers like less or more and how to navigate them. Once I understood how they worked, I faced another challenge: interacting with the Vi text editor to run commands efficiently. This level helped me improve my ability to read and manipulate file contents in restricted environments

- Level32: The challenge here involved using a limited shell environment to my advantage. I experimented with various techniques, including using the tr command to manipulate input and output, and combining commands with pipes to execute scripts. This level taught me how to creatively leverage shell commands to overcome restrictions.
