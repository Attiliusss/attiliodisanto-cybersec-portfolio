# Bandit Levels: Practical Linux & Security Skills

This folder summarizes my experience completing the Bandit wargame from OverTheWire. The project helped me practice Linux command-line skills and improve problem-solving in restricted environments.


## Objective

The goal of this project was to learn and strengthen skills in directory navigation, file manipulation, shell usage, and command-line tools by solving progressively challenging levels.


## Level groups

All Bandit levels have been organized into five groups by type:

- Levels 0–12
- Levels 13–18
- Levels 19–24
- Levels 27–31
- Levels 25 and 32

For each group, I included the commands I actually used during the challenges, focusing on the key tools and techniques rather than full step-by-step solutions.


## Environment

All challenges were completed in a Linux environment using the shell.


## Skill learned

- Navigating directories
- Viewing and manipulating files
- Using command-line tools
- Working with shell scripts and pipelines
- Understanding limited shell environments and pagers
